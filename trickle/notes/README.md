# Prime Refrigeração e Climatização - Web App

Este é o site institucional desenvolvido para a **Prime Refrigeração e Climatização**, uma assistência técnica especializada no Vale do Itajaí.

## Características do Design
- **Paleta de Cores**: Baseada no logo original (Azul e Amarelo).
- **Estilo**: Moderno, com elementos visuais que transmitem confiança e profissionalismo.
- **Estrutura**: Single Page Application (SPA) com seções focadas em conversão (CTA).
- **Imagens**: Uso de fotos reais fornecidas pelo cliente para gerar autenticidade.

## Seções
- Hero (Impacto inicial)
- Diferenciais (Por que escolher a Prime)
- Serviços (Refrigeradores e Lavadoras)
- Galeria (Fotos reais dos trabalhos)
- Depoimentos (Prova social)
- Contato (Formulário e WhatsApp)

## Regras de Manutenção
- Sempre que houver mudanças na estrutura ou novos serviços, este README deve ser atualizado.
- Manter o padrão de cores e ícones Lucide.
- As imagens devem ser carregadas preferencialmente via assets do Trickle.
